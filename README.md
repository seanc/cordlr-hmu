# cordlr-hmu [![NPM version](https://badge.fury.io/js/cordlr-hmu.svg)](https://npmjs.org/package/cordlr-hmu)

> Cordlr hmu plugin

## Installation

```sh
$ npm install --save cordlr-hmu
```

Then add it to your config.

```js
{
  "plugins": [
    "cordlr-hmu"
  ]
}
```

## Usage

```
hmu <name> [...args]
```
